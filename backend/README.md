# 🚀 Backend - AI Growth Ads  
Ce dossier contient l'API backend de la plateforme **AI Growth Ads**.  
Il gère la communication entre le front-end, la base de données et le module d'intelligence artificielle.

## 🔹 Technologies utilisées  
- ⚙️ **FastAPI** → Framework backend rapide et performant  
- 🛢 **PostgreSQL** → Base de données pour stocker les campagnes publicitaires  
- 🔄 **SQLAlchemy** → ORM pour la gestion des données  
- 🔐 **OAuth2** → Authentification et sécurité des utilisateurs  
- 🔥 **TensorFlow/PyTorch** → Module IA d’optimisation publicitaire
